# HR Analytics Dashboard

This project is an interactive HR Analytics Dashboard created using Power BI.  
It provides insights into employee attrition, employee demographics, department analysis, and job role statistics.

## Dashboard Preview

![HR Analytics Dashboard](https://github.com/chandrabose165/Hr_database/blob/main/Screenshot%202026-05-26%20074136.png)

## Features
- Employee Attrition Analysis
- Department-wise Attrition
- Age Group Analysis
- Gender Distribution
- Job Role Insights
- Education-based Filtering

## Technologies Used
- Power BI
- Data Analytics
- HR Dataset Visualization
